def hooks(self):
Generator for tagHOOK info.

Hooks are carved using the same algorithm, but different starting points for desktop hooks and thread hooks. Thus the algorithm is presented in this function and the starting point is acquired by calling hook_params (which is then sub- classed by tagTHREADINFO.

def windows(self, win, filter=True, level=0):
Traverses windows in their Z order, bottom to top.

@param win: an HWND to start. Usually this is the desktop window currently in focus.

@param filter: a callable (usually lambda) to use for filtering the results. See below for examples:

# only print subclassed windows filter = lambda x : x.lpfnWndProc == x.pcls.lpfnWndProc

# only print processes named csrss.exe filter = lambda x : str(x.head.pti.ppi.Process.ImageFileName).lower() == "csrss.exe" if x.head.pti.ppi else False

# only print processes by pid filter = lambda x : x.head.pti.pEThread.Cid.UniqueThread == 0x1020

# only print visible windows filter = lambda x : 'WS_VISIBLE' not in x.get_flags()
