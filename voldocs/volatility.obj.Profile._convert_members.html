v.o.P._convert_members(self, cname) : method documentation

Part of volatility.obj.Profile

Convert the structure named by cname from the c description present in vtypes into a list of members that can be used for later parsing.

cname is the name of the struct.

We expect the vtypes value to be a list of the following format

[ Size of struct, members_dict ]

members_dict is a dict of all members (fields) in this struct. The key is the member name, and the value is a list of this form:

[ offset_from_start_of_struct, specification_list ]

The specification list has the form specified by self._list_to_type() above.

We return an object that is a CType or has been overridden by object_classes.
