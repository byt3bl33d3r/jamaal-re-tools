v.p.o.w.w._.Parent(self) : method documentation

Part of volatility.plugins.overlays.windows.win2003._MMVAD_SHORT

Return the Vad's parent node, being sure to chop off the lower 3 bits, because _MMADDRESS_NODE.u1.Parent is a packed union with _MMADDRESS_NODE.u1.Balanced. We do not want the Balanced part of the value.

Not chopping off these 3 bits is the reason why our vadtree plugin didn't work since introduction of profiles other than Windows XP.
