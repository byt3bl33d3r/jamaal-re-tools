v.p.s.S.get_reverse_map(self, addr_space, tasks, verbfd=None) : method documentation

Part of volatility.plugins.strings.Strings

Generates a reverse mapping from physical addresses to the kernel and/or tasks

Returns: dict of form phys_page -> [isKernel, (pid1, vaddr1), (pid2, vaddr2) ...] where isKernel is True or False. if isKernel is true, list is of all kernel addresses
