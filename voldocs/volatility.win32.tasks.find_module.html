v.w.t.find_module(modlist, mod_addrs, addr) : function documentation

Part of volatility.win32.tasks

Uses binary search to find what module a given address resides in.

This is much faster than a series of linear checks if you have to do it many times. Note that modlist and mod_addrs must be sorted in order of the module base address.
