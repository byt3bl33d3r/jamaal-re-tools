def v(self):
overrides volatility.obj.BaseObject.v
Use zread to help emulate reading null-terminated C strings across page boundaries.

@returns: If all bytes are available, return the full string as a raw byte buffer. If the end of the string is in a page that isn't available, return as much of the string as possible, padded with nulls to the string's length.

If the string length is 0, vtop() fails, or the physical addr of the string is not valid, return NoneObject.

Note: to get a null terminated string, use the __str__ method.
