def write_callback(option, _opt_str, _value, parser, *_args, **_kwargs):
Callback function to ensure that write support is only enabled if user repeats a long string

This call back checks whether the user really wants write support and then either enables it (for all future parses) by changing the option to store_true, or disables it permanently by ensuring all future attempts to store the value store_false.
