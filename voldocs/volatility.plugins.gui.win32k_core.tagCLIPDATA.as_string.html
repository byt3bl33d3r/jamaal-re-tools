Part of volatility.plugins.gui.win32k_core.tagCLIPDATA

Format the clipboard data as a string.

@param fmt: the clipboard format.

Note: we cannot simply override __str__ for this purpose, because the clipboard format is not a member of (or in a parent-child relationship with) the tagCLIPDATA structure, so we must pass it in as an argument.
