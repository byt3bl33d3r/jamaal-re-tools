v.o.P._get_dummy_obj(self, name) : method documentation

Part of volatility.obj.Profile

Returns a dummy object/profile for use in determining size and offset of substructures. This is done since profile are effectively a compiled language, so reading the value from self.vtypes may not be accurate.
