def _imported_functions(self):
Generator for imported functions.

@return: tuple (Ordinal, FunctionVA, Name)

If the function is imported by ordinal, then Ordinal is the ordinal value and Name is None.

If the function is imported by name, then Ordinal is the hint and Name is the imported function name (or None if its paged).

FunctionVA is the virtual address of the imported function, as applied to the IAT by the Windows loader. If the FirstThunk is paged, then FunctionVA will be None.
